《MySQL必知必会》

连接数据库

  • mysql -h HostName -u UserName -p

查看数据库、表、列

  • SHOW DATABASES;

    SHOW TABLES;

    DESC tablename;

检索数据

  • 检索多列

    SELECT colname1, colname2 FROM tablename;

  • 检索所有列

    SELECT * FROM tablename;

  • 检索不同行(去重)

    SELECT DISTINCT colname1, colname2 FROM tablename;

    注：DISTINCT会修饰其后所有的列名。

  • 限制结果

    SELECT colname FROM tablename LIMIT 5; // 仅返回五行

    SELECT colname FROM tablename LIMIT 3, 5; // 从第四行开始，一共返回五行

    SELECT colname FROM tablename LIMIT 5 OFFSET 3; // 和上面的等价

    注：行的索引从0开始。

排序结果

  • 升序排列(默认ascend)

    SELECT colname FROM tablename ORDER BY colname1, colname2; // 先按照colname1升序排列，colname1相同的再按照colname2升序排列

  • 降序排列(descend)

    SELECT colname FROM tablename ORDER BY colname1 DESC, colname2; // 先按照colname1降序排列，colname1相同的再按照colname2升序排列

    注：DESC 仅修饰其前面一个列。
